Simple Facts API

REST API
Voir sur github

Comme son nom l'indique, Simple Facts API est une API délivrant des informations courtes, des vérités vérifiées. Pour combattre certaines idées reçues, apprendre de nouvelles choses ou réassimiler des connaissances acquises, intégrer la à vos applis ! Les faits sont 'taggés' pour permettre la recherche des informations qui peuvent intéresser vos utilisateurs, ou correpondre au champs d'application de votre création. N'hésitez pas à l'utiliser pour vos mocks également
